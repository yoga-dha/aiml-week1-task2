# AIML Internship Week 1 - Task 2

## Project Title

Student Performance Data Visualization and Insights

## Objective

The objective of this project is to visualize student performance data and identify trends, patterns, and relationships affecting academic performance (GPA).

## Dataset

Student Performance Dataset

## Visualizations Created

1. GPA Distribution (Histogram)
2. Gender Distribution (Countplot)
3. Study Time vs GPA (Scatter Plot)
4. Correlation Heatmap

## Key Findings

* Students who spend more time studying generally achieve higher GPA values.
* Absences may negatively affect academic performance.
* GPA distribution is balanced across the dataset.
* Academic performance is influenced by multiple factors.
* Study habits play an important role in student success.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Files Included

* AIML_Week1_Task2_Data_Visualization.ipynb
* Student_performance_data_.csv

## Conclusion

Data visualization helped identify important factors affecting student performance. Study time, attendance, and support-related variables appear to influence GPA.
